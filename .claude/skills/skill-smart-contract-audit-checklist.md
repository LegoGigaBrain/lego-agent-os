# Skill: Smart Contract Audit Checklist

## Purpose
Reusable checklist for reviewing Solidity/EVM contracts.

## Checklist

- [ ] Understand intended behavior and invariants.
- [ ] Check access control and role management.
- [ ] Look for reentrancy and external call risks.
- [ ] Validate input ranges and assumptions.
- [ ] Check arithmetic assumptions and precision.
- [ ] Review event coverage for state changes.
- [ ] Validate upgrade / migration approach (if any).
- [ ] Ensure tests cover success + failure + edge paths.
