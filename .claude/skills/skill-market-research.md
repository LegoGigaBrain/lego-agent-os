# Skill: Market Research

## Purpose

Conduct market research to understand market size, trends, customer needs, and opportunities. This skill provides frameworks for gathering and analyzing market intelligence.

## Standards Awareness

Before applying this skill, ensure familiarity with:
- `standards/signalos/brand-strategy.md`
- `standards/signalos/tribe-building.md`

---

## Core Philosophy

> "Research reduces risk. The best strategy is informed strategy."

Market research is:
- Evidence over assumption
- Customer voice over internal opinion
- Trends over snapshots
- Actionable insights over data collection

---

## Research Types

### Primary Research

First-hand data collection:

```
QUALITATIVE METHODS:
- Customer interviews
- Focus groups
- Ethnographic observation
- User testing

QUANTITATIVE METHODS:
- Surveys
- A/B testing
- Analytics analysis
- Market sizing studies
```

### Secondary Research

Existing data analysis:

```
SOURCES:
- Industry reports
- Academic research
- Government data
- Competitor information
- News and publications
- Social listening
```

## Research Process

### Step 1: Define Objectives

```
RESEARCH OBJECTIVES:

PRIMARY QUESTION:
[Main thing we need to learn]

SECONDARY QUESTIONS:
- [Question 1]
- [Question 2]
- [Question 3]

SUCCESS CRITERIA:
[What good research looks like]

DECISIONS IT INFORMS:
[What we'll do with findings]
```

### Step 2: Design Methodology

```
METHODOLOGY:

APPROACH: [Qualitative/Quantitative/Mixed]

METHODS:
- [Method 1]: [Why]
- [Method 2]: [Why]

SAMPLE:
- Target: [Who]
- Size: [How many]
- Recruitment: [How we find them]

TIMELINE:
- Design: [Time]
- Fieldwork: [Time]
- Analysis: [Time]
- Report: [Time]

BUDGET:
[Resources needed]
```

### Step 3: Collect Data

```
DATA COLLECTION:

PREPARATION:
- [ ] Instruments ready
- [ ] Sample recruited
- [ ] Team briefed

EXECUTION:
- [ ] Fieldwork complete
- [ ] Quality checked
- [ ] Data organized

DOCUMENTATION:
- [ ] Raw data stored
- [ ] Notes compiled
- [ ] Initial observations captured
```

### Step 4: Analyze Findings

```
ANALYSIS APPROACH:

QUALITATIVE ANALYSIS:
- Coding methodology
- Theme identification
- Pattern recognition

QUANTITATIVE ANALYSIS:
- Statistical methods
- Segmentation
- Correlation analysis

KEY QUESTIONS:
- What patterns emerge?
- What surprises us?
- What confirms/challenges assumptions?
- What are the implications?
```

### Step 5: Report Insights

```
INSIGHT STRUCTURE:

INSIGHT: [Clear statement]
EVIDENCE: [Supporting data]
IMPLICATION: [What it means]
RECOMMENDATION: [What to do]
```

## Quality Checklist

- [ ] Objectives are clear
- [ ] Methodology is appropriate
- [ ] Sample is representative
- [ ] Analysis is rigorous
- [ ] Insights are actionable
- [ ] Recommendations are specific

---

## Anti-Patterns

- Research for research's sake
- Confirmation bias
- Small/biased samples
- Data without insights
- Insights without action
- One-time snapshot vs. ongoing
