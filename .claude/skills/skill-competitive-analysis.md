# Skill: Competitive Analysis

## Purpose

Conduct comprehensive competitive analysis to understand market positioning, identify opportunities, and inform strategic decisions. This skill provides frameworks for competitive intelligence.

## Standards Awareness

Before applying this skill, ensure familiarity with:
- `standards/signalos/brand-strategy.md`

---

## Core Philosophy

> "Know your competition, but compete with yourself."

Competitive analysis is:
- Understanding the landscape, not copying
- Finding whitespace, not following
- Learning from others' mistakes
- Informing differentiation strategy

---

## Competitive Analysis Framework

### 1. Competitor Identification

```
COMPETITOR MAPPING:

DIRECT COMPETITORS:
(Same product, same market)
- [Competitor 1]
- [Competitor 2]
- [Competitor 3]

INDIRECT COMPETITORS:
(Different product, same need)
- [Competitor 1]
- [Competitor 2]

POTENTIAL COMPETITORS:
(Could enter the space)
- [Competitor 1]
- [Competitor 2]

SUBSTITUTES:
(Alternative solutions)
- [Alternative 1]
- [Alternative 2]
```

## Quality Checklist

- [ ] Key competitors identified
- [ ] Profiles are current
- [ ] Comparison is fair and accurate
- [ ] Whitespace is identified
- [ ] Insights are actionable
- [ ] Battlecards are useful

---

## Anti-Patterns

- Analysis paralysis
- Competitor obsession
- Biased assessment
- Outdated intelligence
- Copying instead of differentiating
- Ignoring indirect competition
