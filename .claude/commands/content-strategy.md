# content-strategy

You are running the **Content Strategy** workflow in LEGO Agent OS.

Goal:
Develop a comprehensive content strategy including content mission, pillars, editorial calendar, and measurement framework. This workflow produces a complete content operating system.

Primary Agent:
- @content-strategist

Supporting Agents:
- @brand-strategist (for brand signal alignment)
- @social-media-manager (for platform considerations)
- @marketing-strategist (for campaign alignment)

Skills:
- Content Strategy
- Editorial Calendar
- Social Media Strategy

Standards:
- `standards/signalos/transmission-framework.md`
- `standards/signalos/brand-strategy.md`
- `standards/signalos/vibe-marketing.md`

## Steps

### 1. Clarify Content Scope

Determine what we're strategizing:

| Scope | Description | Typical Output |
|-------|-------------|----------------|
| **Full Strategy** | Complete content system | Content OS document |
| **Pillar Development** | Define content themes | Pillar framework |
| **Editorial Planning** | Build calendar | Calendar + workflow |
| **Audit & Refresh** | Assess and improve | Recommendations |

Ask:
- What content currently exists?
- What's the brand signal?
- Who is the audience?
- What are the business goals?
- What resources are available?
- What channels are active?

---

### 2. Brand Foundation Check

**Goal:** Ensure content builds on solid brand foundation

Verify these exist:
```
BRAND SIGNAL: [What we stand for]
ONLINESS: [What makes us the only]
TRIBE: [Who we serve]
LANGUAGE STACK: [Voice and vocabulary]
ARCHETYPE: [Brand personality]
```

If any are missing, invoke @brand-strategist first.

### 5. Content Pillars

```

TOPICS: [Key themes and keywords]
CONTENT TYPES: [Formats that work]
TARGET %: [Portion of content]

PILLAR 2: [Name]
[Same structure]

PILLAR 3: [Name]
[Same structure]

[Continue as needed]
```

### Pillar Validation

For each pillar, confirm:
- [ ] Connects to brand signal
- [ ] Serves genuine tribe need
- [ ] Has search/discovery potential
- [ ] We have expertise/authority
- [ ] Distinct from other pillars

**Strategy Quality**
- [ ] Rooted in brand signal

- [ ] Mix is balanced

- [ ] Calendar is realistic
- [ ] Workflow is clear

- [ ] Flexibility is built in

- [ ] Metrics align with objectives

- [ ] Review cadence is set
