# visual-identity

You are running the **Visual Identity** workflow in LEGO Agent OS.

Goal:
Design a comprehensive visual identity system including logo direction, color, typography, imagery, and application guidelines. This workflow produces a complete visual identity system.

Primary Agent:
- @design-strategist

Supporting Agents:
- @visual-designer (for execution)
- @brand-strategist (for strategy foundation)

Skills:
- Visual Identity
- Design Direction
- Image Prompts

Standards:
- `standards/signalos/transmission-framework.md`
- `standards/signalos/brand-strategy.md`

---

### 3. Logo System

```

[How brand character shows in mark]

- Abstract / Literal / Metaphorical

STYLE TERRITORY:
- Modern ←→ Classic
- Simple ←→ Detailed
- Bold ←→ Refined
- Playful ←→ Serious
```

## Quality Checklist

**Strategic Alignment**
- [ ] Reflects brand archetype
- [ ] Expresses brand values
- [ ] Differentiates from competitors
- [ ] Appeals to tribe

**System Quality**
- [ ] All elements work together
- [ ] System is flexible and scalable
- [ ] Clear hierarchy established
- [ ] Accessibility requirements met

**Usability**
- [ ] Guidelines are clear
- [ ] Examples are provided
- [ ] Templates are available
- [ ] Anyone can apply it
