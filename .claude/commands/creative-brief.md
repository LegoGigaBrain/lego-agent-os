# /creative-brief

Generate a strategic creative brief for AI-powered visual production.

---

## Agent

@ai-creative-strategist

## Skills Required

- `skill-ai-creative-stack.md`
- `skill-perplexity-research.md`

---

## Workflow

### 1. Understand the Need

Before writing the brief, clarify:
- What is the purpose of this creative?
- Who is the target audience?
- Where will this appear (platform, context)?
- What is the brand context?
- What already exists that we should reference?

If any of these are unclear, ask clarifying questions before proceeding.

### 2. Research (if needed)

Use Perplexity or WebSearch to:
- Understand current trends in the space
- Find visual references and inspiration
- Research platform best practices
- Identify competitor approaches

### 3. Generate the Brief

Create a comprehensive creative brief using this structure:

```markdown
# Creative Brief: [Project Name]

## Overview

**Project:** [Name]
**Date:** [Date]
**Requested by:** [Who]
**Due:** [When]

---

## Objective

### Business Goal
[What business outcome does this support?]

### Creative Goal
[What should this visual achieve? Awareness? Conversion? Engagement?]

### Success Metrics
[How will we measure if this worked?]

---

## Audience

### Primary Audience
[Who specifically are we targeting?]

### What They Care About
[Their pain points, desires, values]

### Where They'll See This
[Context of viewing]

---

## Platform & Format

### Platform
[Instagram/Twitter/Website/Email/Ad/etc.]

### Format
[Post/Story/Hero/Ad/Thumbnail/etc.]

### Technical Specs
- Aspect Ratio: [1:1 / 16:9 / 9:16 / 4:5]
- Dimensions: [exact pixels if known]
- File Format: [jpg/png/webp/mp4]
- Max File Size: [if applicable]

### Deliverables Needed
- [ ] [Format 1]
- [ ] [Format 2]
- [ ] [Additional sizes/formats]

---

## Creative Direction

### Style
[Photorealistic / Illustration / 3D / Minimal / Bold / etc.]

### Mood
[Energetic / Calm / Professional / Playful / Inspirational / etc.]

### Color Palette
- Primary: [color]
- Secondary: [color]
- Accent: [color]
- Notes: [any color guidance]

### Visual References
[Links to inspiration images, similar work, mood boards]

1. [Reference 1 - URL] - [why this is relevant]
2. [Reference 2 - URL] - [why this is relevant]
3. [Reference 3 - URL] - [why this is relevant]

### Composition Notes
[Guidance on layout, focal points, negative space]

---

## Key Message

### Primary Message
[The one thing this should communicate]

### Supporting Messages
[Secondary points if applicable]

### Tone
[How the message should feel]

---

## Requirements

### Must Include
- [ ] [Required element 1]
- [ ] [Required element 2]
- [ ] [Brand element - logo, colors, etc.]

### Must Avoid
- [ ] [Anti-pattern 1]
- [ ] [Anti-pattern 2]
- [ ] [Competitor similarity]

### Brand Guidelines
[Link to brand guidelines or key brand rules to follow]

---

## Production Notes

### Recommended Approach
[Replicate for photorealistic / Glif for stylized / etc.]

### Generation Parameters
- Style: [if known]
- Quality: [standard/high]
- Variations needed: [number]

### Timeline
- Brief approval: [date]
- First options: [date]
- Revisions: [date]
- Final delivery: [date]

---

## Approval

### Approvers
[Who needs to approve this?]

### Review Criteria
[What are we checking for?]

---

## Notes

[Any additional context, constraints, or considerations]
```

### 4. Review and Refine

Before delivering:
- [ ] Is the objective clear and measurable?
- [ ] Is the audience specifically defined?
- [ ] Are platform specs complete?
- [ ] Is creative direction actionable?
- [ ] Are references included?
- [ ] Are requirements and constraints clear?
- [ ] Is the timeline realistic?

---

## Output

Deliver the complete brief in markdown format, ready for:
1. Stakeholder review and approval
2. Handoff to visual-designer or next phase
3. Use as reference during generation
